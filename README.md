# Coffee-Ground-Truth
Students:
- Assima Amangeldina
- Ceyla Kaya

Research Lead:
- Yasemin Ateş
  
This project builds a shareable campus “coffee fortune” app that uses computer vision to recognize crema/cup-residue patterns after espresso and converts the detected pattern into a playful fortune.

<br>
​
A custom image dataset will be collected and labeled in Roboflow, then a model will be deployed and consumed via the Roboflow Hosted Inference API from a Flutter mobile app or web app.
